FinPlatform (in development)
FinPlatform is a comprehensive, Django-based financial dashboard and community hub designed to give users the tools they need to track markets, stay informed on economic events, and discuss trading strategies. Featuring a sleek dark-themed UI, the platform combines real-time market visualization with an integrated AI assistant and a robust community forum.

Key Features
Market Charts: Search and view market data for various symbols (including crypto like BTC/USDT and traditional stocks) with custom charting tools.

Economic Calendar: Track low, medium, and high-impact financial events (like Non-Farm Payrolls and PMI reports) with specific times and currency impacts.

Community Forum: A fully featured discussion board where users can create threads, categorize posts by market symbol or topic, and engage with other traders. Includes features for pinned threads and impact badges.

AI Assistant: A built-in, chat-based AI service to help users analyze data, ask financial questions, or get platform support.

Secure Authentication: Complete user registration, login, and session management system seamlessly integrated into the platform's UI.

Built With
Backend: Python & Django (Routing, Views, Authentication)

Database: SQLite (Development)

Frontend: HTML5, CSS3 (Custom Dark Theme), and Vanilla JavaScript

Integrations: AI Chat Service, Dynamic Charting
